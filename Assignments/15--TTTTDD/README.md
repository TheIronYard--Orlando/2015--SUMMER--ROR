## 15 - Tic-Tac-Toe, via TDD

Shall we play a game?

##Assignment
```Markdown
* **Yak Shaving**
  * _WIP Issue_: `15 -- TTTTDD -- YOUR NAME`
    * [ ] link to PR in `TIY-Assignments`
  *  _WIP Branch_:
    * [ ] `TIY-Assignments:15--TTTTDD`
  * _WIP Files_:
    * `TIY-Assignments`
      * [ ] `tdd/tic_tac_toe/game_test.rb`
      * [ ] `tdd/tic_tac_toe/game.rb`
* **The Game**
  * [ ] create TIY-Assignments branch named `15--TTTTDD`
  * [ ] create a new folder in that branch at `tdd/tic_tac_toe`
  * [ ] push back to your GitHub account
  * [ ] open PR _from_ `15--TTTTDD` _to_ `master`
  * [ ] plan out what your code needs to be able to do
    * [ ] display a board
      * [ ] at start of game
      * [ ] after each move
    * [ ] have 2 players 
      * [ ] who take turns
      * [ ] place X or O on board
        * [ ] only in open spaces
    * [ ] declare end of game
      * [ ] identify tie
      * [ ] identify winner
        * [ ] 3 in same row
        * [ ] 3 in same column
        * [ ] 3 in same diagonal
  * ** Beast Mode **
    * [ ] make the game work from command line as opposed to in irb
    * [ ] let the person who starts the game choose to have human or computer opponent
    * [ ] make computer opponent play intelligently
  * ** Nightmare Mode **
    * [ ] wrap your game in a Rails app
    * [ ] have your computer play vs itself and learn a valuable lesson abt global thermonuclear war

```
